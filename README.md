# 信号 · Semaphore

> 暗夜中的信号弹，写给同频者的私密连接与精神共鸣

一款面向女性的私密创作与共鸣信号系统。
不是公开内容平台，而是一个安静、安全、充满仪式感的精神空间——让创作者发出信号，让懂得的人看见、回应、彼此照亮。

---

## 项目定位

Semaphore 并非传统的写信或投稿平台，而是一套**女性向私密信号系统**。
创作者发出信号弹，同频的读者看见信号弹、回应信号弹，在低压力、高隐私的环境中完成深度精神连接。

---

## 视觉与氛围设计

### 色彩体系
- 背景色：深紫蓝 `#0F0F1A` / `#1A1A2E`
  静谧、柔和、不刺眼，提供一个可以安心停留的私人暗夜空间。
- 文字色：暖调纸白 `#E8E4F0`
  柔和易读，接近纸张质感。
- 信号色：琥珀金 `#C4A85A`
  代表信号弹发射、被看见的瞬间。
- 共鸣色：淡紫色 `#9B7FD4`
  代表被回应、被接住、精神共振的时刻。

### 交互氛围
- 信号弹在首页并非以列表卡片展示，而是一个个从暗处缓缓上浮，如同从深海升起。
- 读者提交回答后，不会出现“提交成功”类提示，文字安静消失，进入等待状态。
- 被作者选中时，页面柔和亮起，如同远方有人为你点亮一盏灯。
- 阅读时间窗口采用安静的淡化消失机制，而非红色警报式提醒，一道光缓缓收窄、最终合上。

## 核心运行机制

### 作者流程
1. 撰写正文，并发布信号弹（公开引子 + 引导问题）
2. 接收读者的回答并进行筛选
3. 对通过筛选的读者执行授权
4. 读者在限定时间窗口内阅读加密正文
5. 接收读者的反馈、私密礼物或新的信号弹

### 读者流程
1. 看见信号弹 → 提交回答
2. 等待作者筛选与授权
3. 获得权限后在时间窗口内阅读正文
4. 阅读完成后可选择：
   - 作为私密礼物送给作者
   - 公开附在原文下方作为评论
   - 基于自身感悟生成新的信号弹继续传递

### 信号传递链（可溯源）
作者 → 信号弹 → 读者 A → 新信号弹 → 读者 B → 可溯源至原文

每一次共鸣都可追溯来源，形成温和、可持续的女性精神传递网络。

## 技术架构

1. 作者在前端完成创作
2. 使用 Lit Protocol 对正文进行加密
3. 密文存储至 IPFS，CID 记录上链
4. 读者提交回答，内容存入 IPFS 并将 CID 上链通知作者
5. 作者调用合约方法 `approveReader` 授权
6. 链上记录授权状态与过期时间窗口
7. 读者向 Lit 申请解密密钥
8. Lit 校验链上授权信息，条件满足则返回密钥并解密正文
9. 时间窗口到期后，Lit 自动拒绝后续解密请求

---

## 技术栈

- 前端：沉浸式网页应用
- 加密：Lit Protocol（条件解密、时间权限控制）
- 存储：IPFS 去中心化存储
- 链上：智能合约（授权管理、过期策略、权限存证）
- 核心特点：内容主权可控、阅读权限可控、隐私保护、限时访问

## 项目价值

- 极强的隐私保护：内容加密，阅读权限由作者完全掌控
- 低压力精神共鸣：无算法、无流量、无社交内卷
- 链上确权与可控访问：真正实现创作者内容主权
- 仪式感交互设计：回归文字本身，让书写与阅读重新变得珍贵
- 女性向精神共生：以信号为媒介，形成温柔、安全、可延续的共鸣网络
